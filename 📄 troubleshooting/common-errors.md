# Common Errors

## Following Error

Cause:
- Improper PID tuning
- Mechanical binding
- Encoder scaling mismatch

Solution:
- Verify encoder resolution
- Retune PID
- Inspect mechanical system

## EtherCAT Communication Error

Cause:
- Incorrect wiring
- Node configuration mismatch

Solution:
- Verify cable continuity
- Check device IDs
